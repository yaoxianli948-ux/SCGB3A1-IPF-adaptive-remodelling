# SCGB3A1-IPF-adaptive-remodelling

Analysis code for:

**Cross-scale transcriptomic analysis defines a SCGB3A1-associated epithelial adaptive-remodelling axis in idiopathic pulmonary fibrosis**

Yaoxian Li, Tianyi Wu, Jiazheng Zhang, Xiliang Xu
University of Manchester

## Data sources

All datasets are publicly available:

- GSE135893 — Single-cell RNA-sequencing (Habermann et al., Sci Adv 2020)
- GSE32537 — Bulk lung transcriptomics (Yang et al., Thorax 2013)
- GSE195770 — Whole-transcriptome Visium spatial transcriptomics

## Requirements

- Python 3.8+
- scanpy
- statsmodels
- scipy
- pandas
- numpy
- matplotlib
- seaborn

## Analysis code

Code and derived summary tables available upon reasonable request pending manuscript acceptance.
